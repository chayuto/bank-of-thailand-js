# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-04

### Added

- Initial release
- Complete TypeScript/JavaScript client for Bank of Thailand API
- 11 API resource classes (ExchangeRate, DepositRate, LoanRate, etc.)
- Rich analytics features (statistics, change analysis, volatility)
- CSV export functionality (Node.js and browser)
- Universal runtime support (Node.js 20+, Bun 1.3+, browsers)
- Comprehensive error handling
- Full TypeScript 5.7 type definitions
- Zero runtime dependencies
- Modern 2025 development stack (Bun Test, Biome, tsup)
